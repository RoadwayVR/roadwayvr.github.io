Course Outline

This is a starter template. Replace topics with your real content when ready.

Learning Goals

  • Build a simulation-ready road network (nodes, links, lanes)
  • Define traffic demand (movement + volume + speed assumptions)
  • Model signal timing (cycle, splits, lost time)
  • Calibrate and evaluate using KPIs (delay, queue, throughput, emissions)
  • Communicate results with clear plots and tables

Assessment (example)

  • Mini assignments (30%)
  • Midterm project: baseline model (25%)
  • Final project: scenario comparison + report (35%)
  • Participation / quizzes (10%)
